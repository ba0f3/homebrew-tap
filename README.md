## Installation

### Option 1 — Direct install

```bash
brew install ba0f3/tap/<formula>
```

### Option 2 — Tap first, then install

```bash
brew tap ba0f3/tap
brew install <formula>
```

### Option 3 — Use in a `Brewfile`

```ruby
tap "ba0f3/tap"
brew "<formula>"
```
