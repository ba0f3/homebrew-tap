# Ba0f3 Tap

## How do I install these formulae?

`brew install ba0f3/tap/<formula>`

Or `brew tap ba0f3/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ba0f3/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
